The BaitFisher package

BaitFisher can be downloaded from GitHub: The BaitFisher package The BaitFisher package consits of two programs: BaitFisher and BaitFilter.

BaitFisher can be used to designs baits for target DNA enrichment on the basis of multiple sequence alignments that contain contiguous template DNA.

BaitFilter can be used to determine optimal loci for the baits designed by BaitFisher.
